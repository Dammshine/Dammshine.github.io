3.3 Probability Distribution

Describe how likely a random variable or a set of random varaible is to take on each of possible state.

Discrete Variable and Probability Mass Function(PMF)

The PMF maps from a state of a random variable to the probability. eg (P(x=y) )

Joint probability distribution, P(x=x, y=y).

Satisfy following

  • Domain of P must be the set of all possible states of x.
  • Individual value is between 0 and 1.
  • Sum of value = 1.
Continuous Variable and Probability Density Function

Satisfy following

  • Domain of P must be the set of all possible states of x.
  • Individual value is between 0 and 1.
  • The area of curve is 1.
Marginal Probability Distribution

When the probability distribution among few variables, and want to focus on a set of them.

The sum rule

  • For all x, P(x=x) where x is fixed = sum varies of y of P(x=x, y=y).
Conditional Probability

P(y=y | x=x), given x = x, calculate P(y=y).

Independence and Conditional Independence

Two variables says independence if two variable can be expressed as product of two factor. Two variables says conditional independence when given a variable z, x and y are independent.

Expectation, Variance and Covariance

Expectation is the average, mean value.
Variance gives a measure how much the value will vary.
